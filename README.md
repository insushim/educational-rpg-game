# 🎮 Educational RPG Game

**1990년대 리니지 스타일의 3D 교육용 영어 학습 RPG**

## 📖 프로젝트 소개

아이들이 재미있게 영어 단어를 학습할 수 있는 교육용 RPG 게임입니다.
몬스터를 처치하면 영어 퀴즈가 나오고, 정답을 맞추면 골드와 아이템을 획득할 수 있습니다.

## ✨ 주요 기능

- 🗡️ **3D 액션 전투**: 리니지 스타일의 쿼터뷰 전투 시스템
- 📚 **영어 퀴즈 시스템**: 몬스터 처치 시 영어↔한글 퀴즈
- 💰 **보상 시스템**: 
  - 골드 (항상 획득, 랜덤 수량)
  - 아이템 (확률적 드롭)
- 🎒 **인벤토리 시스템**: 아이템 관리
- 📊 **학습 진도 추적**: 학습한 단어 기록

## 🏗️ 프로젝트 구조

```
Assets/
├── Scripts/
│   ├── Player/          # 플레이어 컨트롤러, 전투
│   ├── Enemy/           # 몬스터 AI, 스탯
│   ├── Quiz/            # 영어 퀴즈 시스템
│   ├── Inventory/       # 인벤토리, 아이템
│   ├── Managers/        # 게임 매니저들
│   └── UI/              # UI 컨트롤러
├── Data/
│   ├── Vocabulary/      # 영어 단어 데이터베이스
│   ├── Items/           # 아이템 데이터
│   └── Monsters/        # 몬스터 데이터
├── Prefabs/
├── Scenes/
└── Resources/
```

## 🚀 시작하기

### 필수 요구사항
- Unity 2021.3 LTS 이상
- C# 기본 지식

### 설치 방법
1. 이 저장소를 클론합니다
```bash
git clone https://github.com/insushim/educational-rpg-game.git
```

2. Unity Hub에서 프로젝트를 엽니다

3. 첫 씬을 열고 플레이합니다
